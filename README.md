# oclip-client

[open clipboard] 远程剪切板客户端

Server Website:  https://oclip.hanxi.info/

## Build

- In windows

```
cd platform\win
build.bat
```

## Develop

```
run.bat
```

## Requirement

- In windows: VS2017 (C/C++)
